# Jazz Events Website 🎷✨

Yo! Welcome to the official repository for the **Jazz Events** website. We’re basically building this to make event planning way less of a headache for everyone. Whether it’s a grand debut, a dream wedding, or just a really good party with catering, this site is where it all happens.

## What’s the vibe?

Initially, this website was made to help people in Imus (and beyond!) book their events without the stress. We wanted to create something that looks premium but feels easy to use.

### Cool stuff you can do:

- **Browse Services:** Check out what we offer—from Debut and Weddings to Catering and general Celebrations.
- **Easy Booking:** You can basically pick a date and start planning your big day in a few clicks.
- **Book Now, Pay Later:** This is the big one. We know events are expensive, so we built in a way for users to pay over 6 months. super helpful!
- **Dashboards:** We’ve got spots for users to check their booking status and a staff side for the team to manage everything.
- **Reviews:** See what other people are saying so you know it's legit.

## Tech Stuff

We kept it classic with:

- **Frontend:** HTML5, Vanilla CSS (for those custom gold vibes), and JavaScript.
- **Icons/Fonts:** FontAwesome 6 and some classy Google Fonts (Great Vibes & Montserrat).
- **Backend:** PHP & MySQL (running on XAMPP).

## Why we made this

Honestly, we just want to help transform ordinary moments into something extraordinary. Planning an event shouldn't feel like a chore, so we made a platform that handles the heavy lifting!
